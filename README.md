# sepAI
Open Source to pick your favourite AI Model.


# Getting Started
Prerequisites
1. Python 3.8+ (for Backend)

2. Node.js 16+ (for Frontend)

# Run FastAPI (Backend)
1. `cd backend`
2. `virtualenv venv`
3. `source venv/bin/activate`
4.  `uvicorn main:app --reload`

# Run Nextjs App Route Typescript (Frontend)
1. `cd frontend`
2. Run `npm run dev`
